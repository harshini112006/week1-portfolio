# Personal Portfolio Website

# What the Project Is
This project is a 3-page static personal portfolio website developed as part of a weekly web development task.  
It highlights my academic background, technical interests, projects, and involvement in student organizations using a clean and professional design.
The website follows semantic HTML structure and consistent styling, and it is deployed using GitHub Pages.

# Features
- Three static pages: Home, About, and Contact
- Clean and consistent navigation bar and footer
- Responsive layout suitable for desktop viewing
- Projects section displayed using a card layout
- FAQ section with accordion-style interaction
- Contact form with built-in HTML validation
- Consistent purple and beige color theme
- Simple dark mode toggle button UI (non-functional)
- Deployed live using GitHub Pages
- 
 # Technologies Used:
- HTML5 (Semantic HTML)
- CSS3 (Flexbox and Grid)
- JavaScript (for FAQ accordion interaction)
- Git & GitHub
- GitHub Pages for deployment
